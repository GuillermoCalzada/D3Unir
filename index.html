Objetivo del sitio

Este sitio se creó como práctica para trabajar con Gráficos D3

Los datos muestra se obtuvieron de https://www.kaggle.com/sakshigoyal7/credit-card-customers. Este dataset contiene información de tarjetahabientes y el uso que dan a sus tarjetas de crédito.

Elegí este conjunto de datos que me pareció interesante explorar un poco el comportamiento y grupos de usuarios de tarjetas de crédito, con ello pude validar algunas teorías que tenía en mente referente a estos instrumentos: Usé diferentes gráficos que creo pueden dar claridad del punto que intento

Las base de slo gráficos se obtubieron de diferentes fuentes, al pie de cada gráfico menciono al fuente principal de cada uno
